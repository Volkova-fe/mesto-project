# Проект: Место
Адаптивная верстка веб-страниц
Функционал:
 - Редактирование данных профиля
 - Добавление карточек
 - Удаление карточек
 - Лайк карточки
 - Плавное открытие и закрытие модального окна

* JS
* Figma
